Getting Started
===============

OSX
---

* install Docker
    * https://docs.docker.com/install/
* install go
    * https://golang.org/dl/
* install kind
    ```
    go get sigs.k8s.io/kind
    ```
* create cluster
    ```
    kind create cluster --name=kind-demo
    ```
